# Video Editor Portfolio Website

A modern, aesthetic video editor portfolio website built with Next.js, featuring smooth animations and a professional showcase of video editing work.

## Features

- **Modern Design**: Clean, professional aesthetic with gradient accents
- **Responsive Layout**: Optimized for all device sizes
- **Smooth Animations**: Framer Motion powered animations
- **Video Portfolio**: Dedicated section to showcase edited videos
- **Services Section**: Highlight video editing services
- **Glass Morphism Effects**: Modern UI elements with backdrop blur
- **Performance Optimized**: Built with Next.js 14 and TypeScript

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React Icons

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Customization

- Update video portfolio items in the `videos` array
- Modify services in the `services` array
- Customize colors in `tailwind.config.js`
- Add your own images and content
